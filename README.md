# guia-energia-fotovoltaica-notebooklm
Desafio de Projeto DIO para criação de um notebooklm. O notebook criado como projeto de estudo serve como um guia abrangente sobre a geração doméstica de energia elétrica por meio da fonte solar, reunindo informações técnicas, regulatórias e ambientais fundamentais para entender esse setor no Brasil
